# Blinkit SQL Sales Analysis Project

## 📌 Project Overview
Performed SQL-based analysis on Blinkit's retail sales dataset to extract business insights such as item performance, outlet trends, and customer ratings.

## 🧰 Tools Used
- SQL Server
- SQL Queries: SUM, AVG, COUNT, PIVOT, CASE, Window Functions

## 🔍 Key Insights
- Total Sales by Item Fat Content and Type
- Percentage of Sales by Outlet Size
- Sales Trends by Establishment Year
- Used PIVOT to analyze Fat Content vs Outlet Location

## 📂 Files Included
- `blinkit_queries.sql`: SQL queries used in this project
- `dataset_description.txt`: Basic info about the dataset
